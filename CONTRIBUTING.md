# 贡献指南

感谢您对 RustChain 项目的兴趣！我们欢迎并感谢任何形式的贡献。

## 如何贡献

### 1. 报告问题
- 使用 GitHub Issues 报告 bug 或功能请求
- 请详细描述问题和复现步骤
- 包含相关的环境信息

### 2. 提交代码
1. Fork 本仓库
2. 创建功能分支 (`git checkout -b feature/amazing-feature`)
3. 进行更改并提交 (`git commit -m 'Add some amazing feature'`)
4. 推送分支 (`git push origin feature/amazing-feature`)
5. 创建 Pull Request

### 3. 代码规范
- 遵循项目现有的代码风格
- 确保代码通过测试
- 更新相关文档

## 开发环境设置

```bash
# 克隆仓库
git clone https://github.com/zhuzhushiwojia/Rustchain.git
cd Rustchain

# 安装依赖
pip install -r requirements.txt

# 运行测试
./test-api.sh
```
